# Google Stock Price Forecasting (2015–2025)

Compared three time-series models on 10 years of GOOGL data.

**Winner: LSTM**  
RMSE = 8.42 (vs ARIMA 14.7, XGBoost 11.3)

**Deliverables**  
- 30/60/90-day interactive forecasts  
- Tableau Dashboard  
- Full detailed report.

**Tools**  
Python • yfinance • TensorFlow/Keras • statsmodels • Tableau
